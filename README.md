# **Invoicing System - Invoicer**

* * *

Web application that allows manage invoices in easy way.

## **Requirements**

JDK >= 8
Maven >= 2.5.1

## **Quickstart**

 To execute Invoicer simply type in the command line :
`mvn exec:java`
